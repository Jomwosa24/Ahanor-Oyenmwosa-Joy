Tags Definition
b It defines bold text without any extra importance.
Str It defines text with strong importance.The content inside is displayed in bold.
i Defines a part of text in an alternate voice or mood. The content inside is typically displayed in italics.
em Defines emphasized text. The content inside is displayed in italics.
small Defines smaller text.
mark Specifies text that should be marked or highlighted.
del It specifies text that has been deleted from a document. Browsers will usually strike a line through deleted text.
ins Specifies a text that has been inserted into a document. Browsers usually underline inserted text.
sub Defines a subscript text. Subscript text appears half a character below the normal line and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas.
sup Defines a superscript text. Supercsript text appears half a character above the normal line and it is sometimes rendered in smaller fonts.
